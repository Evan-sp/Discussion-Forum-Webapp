## Setup and run
### Clone repo
```
git clone https://github.com/Evan-sp/Discussion-Forum-Webapp.git
cd discussion-forum-webapp
```
### Install and run
```
npm install
npm run dev
```
### Start server
```
cd server
node server.js
```
